#  钱包模型

## Balance 用户余额

### 模型定义

| 字段          | 类型   | 含义           | 是否私有 | 读/写 |
| ------------- | ------ | -------------- | -------- | ----- |
| id            | Int    | 用户唯一标识符 |          | R     |
| uid           | FK     | 对应用户         |        | R   |
| balance       | Float  | 用户余额       |          | R     |

--------



## TransactionDetail 余额明细

### 模型定义

| 字段          | 类型   | 含义           | 是否私有 | 读/写 |
| ------------- | ------ | -------------- | -------- | ----- |
| id            | Int    | 用户唯一标识符 |          | R     |
| create_time   |Long    | 产生的时间戳   |          | R      |
| transaction_type| Int  | 明细类型       |         | R         |
| amount        | float  |  金额数目        |       | R     |
| uid           | FK     | 对应用户         |        | R   |
| oid           | FK      | 对应订单       |          | R     |

*注释*

transaction_type

| 值 | 含义|
|----|----|
| 0  | 余额收到款项 |
| 1  |用户发起提现申请|
| 2 | 用户提现成功|
| 3 | 用户提现失败，返回到余额   |
